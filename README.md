



# 📱 WhatsApp Automation

A Python tool to send WhatsApp messages automatically.

## Features
- 📩 Send messages manually
- 📁 Send to CSV contacts (bulk messaging)
- ✅ Confirm before sending
- ⏰ Schedule message timing
- 📅 Date/Time in history
- 📜 View message history
- 🗑️ Clear message history
- ⚠️ Error handling

## Requirements
pip install pywhatkit pandas

## How to Use
1. Run the program
2. Choose option 1 for manual message
3. Choose option 2 for CSV bulk message
4. Enter message and time
5. Confirm sending
6. Message sent automatically! ✅


## CSV Format
Name,Phone
John,+919XXXXXXXXX
Rahul,+919XXXXXXXXX

## Important
- WhatsApp Web must be logged in
- Chrome browser required
- Internet connection required
