


# 📱 WhatsApp Automation

A Python tool to send WhatsApp messages automatically.

## Features
- 📩 Send messages to multiple contacts
- ⏰ Schedule message timing
- 📜 View message history
- 🗑️ Clear message history
- ⚠️ Error handling

## Requirements
```bash
pip install pywhatkit
```

## How to Use
1. Run the program
2. Enter phone number with country code (+91XXXXXXXXXX)
3. Enter message
4. Set time (24hr format)
5. Message will be sent automatically!

## Important
- WhatsApp Web must be logged in
- Chrome browser required
- Internet connection required

## Technologies Used
- Python
- pywhatkit
- os
